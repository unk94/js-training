Exercices JavaScript

Avant de commencer les exercices, il est important d'avoir les notions de bases en Javascript à savoir:
  • les variables et leurs types,
  • les conditions,
  • les boucles,
  • les fonctions,
  • quelques notions de POO (Programmation Orientée Objet),
  • la manipulation du DOM
  • les events,
  • et l'utilisation de l'object XMLHttpRequest ou de l'API Fetch afin de faire des requêtes HTTP.


Autre point d'attention: toujours garder à l'esprit des principes de programmation fondamentaux: DRY (Don't Repeat Yourself) & KISS (Keep It Simple Stupid), aussi bien lors de l'étape de réfléxion que durant l'écriture du code.

Ouvrir le script JS ./script.js dans votre IDE, c'est le seul fichier modifiable. Interdiction de toucher au fichier ./index.html

Exercice 1 - Les variables

  • Créer 3 variables: un Number, un String et un Boolean. Afficher leur valeur dans un console.log()

  • Afficher le type de la variable whatIsMyType dans un console.log()

  • Faire une concaténation en utilisant les variables carBrand, price et currency afin d'écrire la phrase suivante : Une Tesla coûte plus de 40000 € tu sais !, et afficher le résultat dans un console.log()

Exercice 2 - Les conditions, les boucles & les fonctions

  • Selon la valeur de la variable johnAge, créer une autre variable legalAge afin d'avoir l'information pour savoir si John est majeur ou non.

  • Parcourir le tableau ages afin de trouver le nombre d'adultes et de renseigner le résultat dans la variable numberOfAdults

  • Créer la fonction howManyAdults(ages), qui prend en paramètre un tableau ages, et renvoie le nombre d'adultes (personnes majeures)

Exercice 3 - POO

  • Créer au moins deux instances valides de la class Burger, et afficher le résultat de la méthode describe() dans un console.log()

  • Implémenter une nouvelle méthode getPrice() dans la classe Burger, sachant que: Un burger classique coûte 8€, un burger veggie coûte 1€ de moins, et le prendre avec des frites ajoute 2€.

  • Afficher, dans un console.log(), la phrase suivante: Le DOM est prêt. après être bien certain que le DOM ait été chargé entièrement.

  • Incrémenter un compteur au click sur l'élément .btn-counter, et afficher ce résultat dans l'élément HTML .btn-counter-value
